# 文件上传系统

一个基于Vue.js和Node.js的现代化文件上传系统，支持拖拽上传、多文件处理和实时进度显示。

## 功能特性

- 🎯 **多种上传方式**: 支持点击选择和拖拽上传
- 📁 **多文件支持**: 可同时上传多个文件
- 📊 **实时进度**: 显示文件上传进度和状态
- 🎨 **现代化界面**: 响应式设计，美观易用
- 💾 **真实存储**: 文件真实保存到服务器
- 🗂️ **文件管理**: 支持删除单个文件或清空所有文件
- 📱 **文件预览**: 根据文件类型显示相应图标

## 技术栈

- **前端**: Vue 3 + Tailwind CSS
- **后端**: Node.js + Express
- **文件上传**: Multer
- **样式**: Tailwind CSS

## 安装和运行

### 1. 安装依赖

```bash
npm install
```

### 2. 启动服务器

```bash
npm start
```

或者使用开发模式（需要安装nodemon）：

```bash
npm run dev
```

### 3. 访问应用

打开浏览器访问: http://localhost:3000

## API 接口

### 上传文件
- **POST** `/api/upload` - 单文件上传
- **POST** `/api/upload-multiple` - 多文件上传

### 文件管理
- **GET** `/api/files` - 获取文件列表
- **DELETE** `/api/files/:filename` - 删除指定文件
- **DELETE** `/api/files` - 清空所有文件

### 静态文件
- **GET** `/uploads/:filename` - 访问已上传的文件

## 特性说明

### 文件存储
- 所有上传的文件保存在 `uploads/` 目录下
- 文件名会自动添加时间戳和随机数以避免重名
- 支持最大100MB的文件上传

### 安全性
- 文件大小限制：100MB
- 基本的文件类型验证
- CORS跨域支持

### 响应式设计
- 适配桌面和移动设备
- 拖拽区域有视觉反馈
- 上传进度实时更新

## 开发说明

如需修改配置，可以编辑 `server.js` 中的相关设置：

- 端口号（默认3000）
- 上传目录（默认./uploads）
- 文件大小限制（默认100MB）
- 支持的文件类型

## 注意事项

- 确保Node.js版本 >= 14.0
- 上传目录需要有写入权限
- 大文件上传可能需要较长时间
